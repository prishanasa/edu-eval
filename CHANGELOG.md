# Changelog - EduEval AI

## v1.0.0 (March 2026)
### Added
- RAG pipeline with FAISS vector database
- Sentence Transformer embeddings (all-MiniLM-L6-v2)
- Groq LLaMA-3.3 70B integration for answer evaluation
- PDF upload and indexing
- Score gauge visualization
- Concept coverage and gap analysis
- Detailed feedback and model answer generation
- Batch evaluation via CSV upload
- Dark theme Streamlit UI
- Persistent FAISS index storage
- Input validation and error handling
- Evaluation logging utility
- Centralized configuration file
- Unit tests for core functions

- Streamlit UI with sidebar configuration
